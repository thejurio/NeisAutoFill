# UX 개선 로드맵 (v1.4.0)

작성: 2026-07-17 (리팩토링 완료 직후) · 상태: ⬜ 대기 · 🔄 진행중 · ✅ 완료
배경: 선생님 작업 여정(명단·계획 → 등급 → 서술문 → 나이스 연결 → 자동 입력) 기준 UI/UX 감사 결과.
분석 원문·근거는 세션 기록 참고. **원칙: 기능 변화 없음 — 표면(레이아웃·위계·안내)만 개선.**

---

## 1. 확정된 개선안 (사용자 승인: 전부 진행)

| # | 항목 | 내용 | 효과 |
|---|---|---|---|
| U1 | 빈 화면 시작 안내 | 자료 없을 때 성적표 자리에 3단계 카드(① 명단·계획 [시작하기] → ② 성적 → ③ 서술문) + "엑셀을 끌어다 놓아도 됩니다". 자료 로드되면 자동 소멸 | 첫 실행 진입 장벽 해소 (최대 효과) |
| U2 | 로그 상태바화 | 기본 = 한 줄 상태바(마지막 로그 + [▲ 자세히]). 펼치면 기존 전체 로그. ⚠/✗ 발생 시 상태바 색 변경(주황/빨강)으로 클릭 유도 | 위압감 제거 + 성적표 세로 170px 확대 |
| U3 | 입력 버튼 연결 게이팅 | 나이스 미연결이면 [이 과목/전과목 입력] 비활성 + 툴팁 "나이스 연결 후 사용할 수 있습니다" | 오류 팝업 경로 제거 |
| U4 | 탭 안 한 줄화 | "영역: ..." 중복 텍스트 제거, 행/열 선택 힌트 → ⓘ 툴팁. 일괄 입력(좌) + 성취기준·입력 버튼(우) 한 줄 배치 | 성적표 한 줄 확대 + 정돈 |
| U5 | NEIS 접속 정리 | "①" 제거 → 🌐 NEIS 접속. **연결되면 버튼 숨김**(🟢 배지만), 끊기면 재등장 | 거짓 단서 제거, 필요한 것만 표시 |
| U6 | 툴바 흐름 재배치 | 흐름 그룹 [📝 명단·계획][✨ 교과학습] │ 파일 그룹 [📂 성적파일▼][📤 내보내기] | 작업 여정과 버튼 순서 일치 |
| U7 | 글꼴 소폭 상향 | 툴바·버튼 12.5→13px, 로그(펼침) 11.5→12px | 교사 연령층 가독성 |

### 결정사항
- U2 기본값 = **접힘** (사용자 확인). 상태는 state.json 에 저장해 재시작 후 유지.
- 버튼 위계 원칙: 화면당 Primary(파랑) 는 "다음 단계로 나아가는 액션" 하나만.
- 자료가 로드된 화면(레이아웃 자체)은 유지 — 배치·위계·안내만 손본다.

## 2. 진행 순서

**U4 → U5 → U6 → U3 → U7 → U2 → U1** — 작은 정리(한 줄화·버튼)부터 큰 구조(상태바·빈 화면) 순.
각 단계: 빌드·테스트 90개 + 부팅 + 스크린샷 확인. U1·U2 는 캡처를 사용자 확인 후 다음 진행.
전부 끝나면 사용설명서 스크린샷 재촬영(가짜 샘플 데이터로) + v1.4.0 릴리스 (리팩토링 포함).
