준비

설치는 압축 풀고 더블클릭, 끝

  1. NeisAutoFill-win-x64.zip 우클릭 → 압축 풀기
  2. 폴더 안 NeisAutoFill.App.exe 더블클릭
  3. 새 버전이 나오면 실행 시 자동 안내 → [예]만 누르면 알아서 업데이트·재시작
💡 모든 자료는 내 문서\NeisAutoFill 폴더에 엑셀로 자동 저장됩니다.
프로그램을 다시 켜면 마지막 자료가 그대로 열립니다.
화면 구성

메인 화면, 이 다섯 가지만 알면 됩니다

학생 이름은 모두 가상의 예시입니다
  • 왼쪽 위 — 자료: 📝 명단·계획 📂 성적파일 📤 내보내기 ✨ 교과학습
  • 오른쪽 위 — 나이스: ① NEIS 접속 + 🔴/🟢 연결 상태, ⚙ 설정
  • 과목 탭 — 국어·수학… 클릭해서 전환
  • 성적표 — 초록=잘함 · 노랑=보통 · 빨강=노력요함 · 주황 – = 미입력
  • 아래 검은 창 — 작업 기록. 문제가 생기면 이 내용을 복사해 문의하세요
STEP 1

학생 명단 — 복사 → 붙여넣기 한 번

[📝 명단·계획] 창 — 왼쪽이 명단, 오른쪽이 평가계획
  1. 엑셀에서 번호·이름 열 복사 (Ctrl+C)
  2. 📝 명단·계획 열고 📋 붙여넣기 클릭
  3. 끝 — 명단 전체가 한 번에 입력됩니다
  • 이름만 붙여넣어도 번호 자동 부여
  • 직접 입력도 가능 — 이름 쓰면 번호가 자동으로
  • 학생 삭제 시 뒷번호 자동 당김
STEP 2

평가계획 — 파일만 넣으면 AI가 읽습니다

  1. [📝 명단·계획] 창에서 📄 평가계획 불러오기
  2. 평가계획서 파일 선택 — 이지에듀·스쿨마스터 등 양식 무관PDF · HWP · HWPX · 엑셀 모두 지원 (창에 끌어다 놓아도 됨)
  3. AI가 과목별로 인식 (1~3분) → 표에 자동으로 채워짐
  4. 내용 확인·수정 후 저장 후 적용
💡 저장하면 성적표가 자동으로 만들어집니다 — 과목 탭·영역·학생까지. 양식을 따로 만들 필요가 없습니다.
⚠ 영역 이름이 나이스 화면과 같은지 꼭 확인하세요. 자동 입력의 기준이 됩니다.
STEP 3

등급 입력 — 드래그하고 버튼 한 번

  • 셀 두 번 클릭 → 등급 목록에서 선택 (기본)
  • 여러 칸 드래그 → 표 위 잘함 보통 노력요함 버튼으로 일괄 입력키보드 1 2 3도 동일 · Del = 지우기
  • 이름 클릭 = 그 학생 전체 선택 · 영역 제목 클릭 = 그 영역 전체 선택Ctrl+클릭으로 선택 추가
  • 엑셀 범위 복사 → 표에서 Ctrl+V 붙여넣기
  • 실수하면 Ctrl+Z — 일괄 작업도 한 번에 취소
  • 입력 후 2초 뒤 자동 저장 — 저장 버튼 없음
💡 추천 요령: 영역 제목 클릭 → [잘함] 전원 입력 → 예외 학생만 고치기
STEP 3+

성취기준 보면서 입력하기

[📖 성취기준]을 켜면 창이 옆으로 넓어지며 기준이 나타납니다
  • 성적표 오른쪽 위 📖 성취기준 버튼 클릭
  • 현재 과목의 영역별 · 등급별 평가기준이 오른쪽에 표시
  • 과목 탭을 바꾸면 내용도 함께 바뀜
  • 다시 누르면 닫히고 창 크기 원상 복구
자료 관리

모든 자료는 엑셀과 자유롭게 오갑니다

  • 자동 저장 — 성적·평가계획은 내 문서\NeisAutoFill 폴더의 엑셀 파일에 실시간 저장
  • 📤 내보내기 — 성적표를 원하는 위치에 엑셀 사본으로 저장 (공유·보관용)
  • 엑셀에서 직접 고쳐도 됨 — 다음 실행 때 그대로 반영
  • 📂 성적파일 — 다른 엑셀 열기 · ▼ 버튼 = 최근 파일 목록
  • 서술문도 동일: 💾 저장 / 📤 내보내기 / 📂 불러오기 (교과학습 창)
STEP 4

AI 서술문 — 전과목 자동 생성

[✨ 교과학습] 창 — 생성된 서술문은 그 자리에서 바로 수정
  1. ✨ 교과학습 클릭
  2. 🌐 전과목 생성 → 과목 체크 → 시작
  3. 생성은 백그라운드 진행 — 창을 닫아도 계속, 메인 아래에 진행 표시
  4. 완성된 서술문은 칸을 클릭해 자유롭게 수정
💡 글자 수·말투를 바꾸려면 ⚙ 설정 → [AI 서술문] (13장 참고)
STEP 5

나이스 연결 — 로그인만 하면 자동

  1. ① NEIS 접속 클릭 → 나이스 전용 브라우저가 열림
  2. 그 브라우저에서 선생님이 직접 로그인프로그램은 로그인에 절대 관여하지 않습니다
  3. 나이스에서 [성적 → 교과별 평가] → 학년·반·과목 선택 → [조회]
  4. 몇 초 안에 🔴 미연결 → 🟢 연결됨 자동 전환 — 끝
⚠ 반드시 ① NEIS 접속 버튼으로 연 브라우저를 쓰세요. 평소 쓰던 창에는 연결되지 않습니다.
💡 처음 한 번, ⚙ 설정 → [일반]에서 우리 시·도 교육청 확인
STEP 6

나이스 자동 입력 — 과목 체크하고 시작

과목 선택 창 — 원하는 과목만 골라서
등급나이스 [교과별 평가] 화면에서 → 성적표의 🚀 전과목 입력
서술문나이스 [학기말 종합의견] 화면에서 → 교과학습 창의 🚀 전과목 입력
  • 과목마다 입력 → 값 검증 → 나이스 [저장] → 다음 과목 자동 반복
  • 검증 실패 시 저장하지 않고 즉시 중단 — 잘못 저장될 걱정 없음
  • 끝나면 과목별 결과 요약 표시
  • 한 과목만: ▶ 이 과목 입력이때는 저장을 자동으로 누르지 않음 — 확인 후 나이스에서 직접 [저장]
⚠ 처음엔 한 과목으로 시험 → 나이스에서 값 확인 → 그다음 전과목
STEP 6+

화면과 자료가 다르면? 확인 창이 지켜줍니다

학생·과목·영역이 나이스 화면과 완전히 같지 않으면, 입력 전에 확인 창이 자동으로 나타납니다. 마음대로 추측해서 입력하지 않습니다.

상황확인 창에서 할 일
과목 이름이 다름노란 경고 확인 → "이대로 입력 동의" 체크해야 진행
학생 이름이 다름 (전입생 등)그 학생 자리에 누구의 성적을 넣을지 목록에서 선택 (비슷한 이름은 자동 추천) 또는 [입력 안 함]
영역 이름·개수가 다름화면에 보이는 순서대로 한 줄씩, 어떤 영역의 성적을 넣을지 선택 — 이름이 같은 영역은 자동으로 골라져 있고, 뺄 줄은 [입력 안 함]
💡 문제가 하나도 없으면 이 창은 나타나지 않고 바로 입력됩니다.
설정

⚙ 설정 — 세 가지만 기억하세요

[AI 서술문] 탭 — 글자 수 · 영역 수 · 말투
  • [일반] — 우리 시·도 교육청 / 자동클릭 속도컴퓨터가 느려 입력이 자주 실패하면 [보통]·[느림]으로
  • [AI 서술문] — 목표 글자 수(예: 250) / 서술문에 담을 최대 영역 수 / 전체 말투 지시문
  • [평가 단계] — 단계 수(2~5)와 이름을 나이스와 똑같이단계별 'AI 서술 뉘앙스'로 등급별 말투도 지정
⚠ 단계 이름이 나이스 드롭다운 글자와 다르면 자동 입력이 안 됩니다 (예: 매우잘함/잘함/보통/노력요함)
