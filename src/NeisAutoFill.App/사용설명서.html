시작

설치하면 첫 화면이 길을 알려줍니다

처음 켰을 때 — 화면 가운데 안내를 따라가면 됩니다
  1. NeisAutoFill-win-x64.zip 우클릭 → 압축 풀기
  2. 폴더 안 NeisAutoFill.App.exe 더블클릭
  3. 첫 화면의 시작하기를 누르면 STEP 1 부터 시작됩니다
  • 새 버전이 나오면 실행할 때 자동 안내 — [예]만 누르면 알아서 업데이트
  • 이미 만든 엑셀(성적·평가계획서)이 있다면 창에 끌어다 놓기만 하면 됩니다
  • 모든 자료는 내 문서\NeisAutoFill 폴더에 자동 저장 — 다음에 켜면 그대로 열립니다
화면 구성

메인 화면, 이 다섯 가지만 알면 됩니다

학생 이름은 모두 가상의 예시입니다
  • 왼쪽 위 — 작업 순서대로: 📁 자료 준비 ✨ 교과학습, 그 옆에 파일 열기·내보내기
  • 오른쪽 위 — 🌐 NEIS 접속 + 🔴/🟢 연결 상태, ⚙ 설정 ?연결되면 접속 버튼은 자동으로 사라집니다
  • 과목 탭 — 국어·수학… 클릭해서 전환
  • 성적표 — 초록=잘함 · 노랑=보통 · 빨강=노력요함 · 주황 – = 아직 미입력
  • 맨 아래 상태줄 — 방금 한 일이 한 줄로 표시. 자세한 기록은 ≡ 로그
STEP 1

자료 준비 ① — 학생 명단은 붙여넣기 한 번

[📁 자료 준비] 창 — 왼쪽이 학생 명단, 오른쪽이 평가계획
  1. 엑셀에서 학생 번호·이름 열 복사 (Ctrl+C)
  2. 📁 자료 준비를 열고 왼쪽 아래 📋 붙여넣기 클릭
  3. 끝 — 명단 전체가 한 번에 들어갑니다
  • 이름만 붙여넣어도 번호가 자동으로 붙습니다
  • 빈 칸에 이름을 직접 입력해도 됩니다
  • 전학 등으로 학생이 바뀌면 언제든 다시 열어 고치세요 — 입력해 둔 성적은 그대로 유지됩니다
STEP 1

자료 준비 ② — 평가계획서는 파일만 넣으면 AI가 읽습니다

  1. 같은 창 오른쪽 위 📄 평가계획 불러오기 클릭
  2. 학교 평가계획서 파일 선택 — 이지에듀·스쿨마스터 등 양식 무관PDF · HWP · HWPX · 엑셀 모두 지원
  3. AI가 과목별로 차례차례 읽습니다 (1~3분, 진행 상황이 화면에 표시)
  4. 표에 채워진 내용을 훑어보고 틀린 곳만 고친 뒤 저장 후 적용
💡 저장하는 순간 과목 탭과 성적표가 자동으로 만들어집니다 — 양식을 따로 만들 필요가 없습니다.
⚠ 영역 이름이 나이스 화면과 같은지 꼭 확인하세요. 자동 입력이 이 이름으로 학생과 영역을 찾습니다.
STEP 2

등급 입력 — 드래그하고 버튼 한 번

  • 셀 두 번 클릭 → 등급 목록에서 선택 (기본)
  • 여러 칸 드래그 → 표 위 잘함 보통 노력요함 으로 일괄 입력키보드 1 2 3도 동일 · Del = 지우기
  • 학생 이름 클릭 = 그 학생 전체 · 영역 제목 클릭 = 그 영역 전체 선택Ctrl+클릭으로 선택 추가
  • 엑셀에 등급이 이미 있다면: 범위 복사 → 표에서 Ctrl+V
  • 실수하면 Ctrl+Z — 일괄 작업도 한 번에 취소
  • 입력하면 잠시 후 자동 저장 — 저장 버튼을 누를 필요가 없습니다
💡 빠른 요령: 영역 제목 클릭 → [잘함]으로 전원 입력 → 예외인 학생만 고치기
STEP 2+

성취기준 보면서 입력하기

[📖 성취기준]을 켜면 창이 옆으로 넓어지며 기준이 나타납니다
  • 성적표 오른쪽 위 📖 성취기준 클릭
  • 현재 과목의 영역별 · 등급별 평가기준이 오른쪽에 표시
  • 과목 탭을 바꾸면 내용도 함께 바뀜
  • 다시 누르면 닫히고 창 크기도 원래대로
STEP 3

AI 서술문 — 전과목 자동 생성

[✨ 교과학습] 창 — 완성된 서술문은 칸을 클릭해 바로 수정
  1. ✨ 교과학습 클릭
  2. 🌐 전과목 생성 → 과목 체크 → 🚀 생성 시작
  3. 생성은 뒤에서 알아서 진행 — 창을 닫아도 계속되고, 메인 아래 상태줄에 진행이 표시됩니다
  4. 끝난 서술문은 읽어보고 자유롭게 고치세요 — 고친 내용은 자동 보관
💡 글자 수·말투를 바꾸고 싶으면 ⚙ 설정 → [AI 서술문] (12장)
STEP 4

나이스 연결 — 로그인만 하면 자동

  1. 오른쪽 위 🌐 NEIS 접속 클릭 → 나이스 전용 브라우저가 열림
  2. 그 브라우저에서 선생님이 직접 로그인프로그램은 로그인에 절대 관여하지 않습니다
  3. 나이스에서 [성적 → 교과별 평가] → 학년·반·과목 선택 → [조회]
  4. 몇 초 안에 🔴 미연결 → 🟢 연결됨으로 바뀌고, 흐리게 잠겨 있던 [입력] 버튼들이 켜집니다
⚠ 반드시 🌐 NEIS 접속 버튼으로 연 브라우저를 쓰세요. 평소 쓰던 창에는 연결되지 않습니다.
💡 처음 한 번, ⚙ 설정 → [일반]에서 우리 시·도 교육청을 확인하세요.
STEP 5

나이스 자동 입력 — 과목 체크하고 시작

과목 선택 창 — 원하는 과목만 골라서
등급나이스 [교과별 평가] 화면에서 → 성적표 오른쪽 위 🚀 전과목 입력
서술문나이스 [학기말 종합의견] 화면에서 → 교과학습 창의 🚀 전과목 입력
  • 과목마다 입력 → 값 확인 → 나이스 [저장] → 다음 과목을 자동 반복
  • 확인에 실패하면 저장하지 않고 그 자리에서 중단 — 잘못 저장될 걱정 없음
  • 끝나면 과목별 결과가 요약되어 나타남
  • 한 과목만: ▶ 이 과목 입력이때는 자동 저장하지 않음 — 화면 확인 후 나이스에서 직접 [저장]
⚠ 처음엔 한 과목으로 시험 → 나이스에서 값 확인 → 그다음 전과목
STEP 5+

화면과 자료가 다르면? 확인 창이 지켜줍니다

학생·과목·영역이 나이스 화면과 완전히 같지 않으면 입력 전에 확인 창이 자동으로 나타납니다. 프로그램이 추측으로 입력하는 일은 없습니다.

상황확인 창에서 할 일
과목 이름이 다름노란 경고를 읽고 "이대로 입력 동의"에 체크해야 진행됩니다
학생 이름이 다름 (전입생 등)"이 학생 자리에 누구의 성적을 넣을지" 목록에서 선택 — 비슷한 이름은 미리 골라져 있음. 넣지 않으려면 [입력 안 함]
영역 이름·개수가 다름화면에 보이는 순서대로 한 줄씩 "이 줄에는 어떤 영역의 성적"을 선택 — 이름이 같은 영역은 자동 선택됨. 뺄 줄은 [입력 안 함]
💡 아무 문제가 없으면 이 창은 나타나지 않고 바로 입력됩니다.

[AI 서술문] 탭 — 글자 수 · 영역 수 · 말투
  • [일반] — 우리 시·도 교육청 / 자동클릭 속도컴퓨터가 느려 입력이 자주 실패하면 [보통]·[느림]으로
  • [AI 서술문] — 목표 글자 수(예: 250) / 서술문에 담을 최대 영역 수 / 전체 말투 지시문
  • [평가 단계] — 단계 수(2~5)와 이름을 나이스와 똑같이단계별 'AI 서술 뉘앙스'로 등급별 말투도 지정
⚠ 단계 이름이 나이스 화면의 글자와 다르면 자동 입력이 안 됩니다 (예: 매우잘함/잘함/보통/노력요함)
